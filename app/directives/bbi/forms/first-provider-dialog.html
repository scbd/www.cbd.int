First Provider Record

Provider profile records you create reference 2 other BBI records, a Contact and an Organization. While creating your provider profile, you will be asked to select an organization and contact. If they do not exist you will have to leave the Provider form and create them.
Procedure for creating a provider profile:

  1. Create the requesting Organization (if it does not exist)
  2. Create the contact person (if it does not exist)
  3. Lastly, complete your provider profile.
