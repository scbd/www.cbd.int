Understanding BBI Records

All BBI records have 3 states. BBI records are approved for publishing by the CBD. Anyone can suggest an edit to a published record.

Some BBI records may reference other BBI records. If a record you are creating needs to reference a record not already avalable in the search. You will have to save your current record and create the reference record. Once the reference record is created you can edit you saved draft and select you new reference record.


  1. Only you can see the record.

  2. You have submitted your record and it is pending approval.

  3. Everyone can see the record.
