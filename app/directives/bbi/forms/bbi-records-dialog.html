Understanding BBI Records

BBI records can be created by any user with a CBD account but must be approved by the CBD Secretariat before they are published. Owners of the BBI records can suggest edits to their published record.

BBI records can reference other BBI records. If a record you are creating needs to reference a record not already available on the BBI web platform you have to save your current record and create the reference record. Once the reference record is created you can edit the saved draft and select your new reference record.

BBI records are presented on your dashboard as drafts, requests pending approval or as published records.


  1. Only you can see the record.

  2. You have submitted your record and it is pending approval.

  3. Everyone can see the record.
