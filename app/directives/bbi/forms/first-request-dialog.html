First Assistance Request Record

Assitance requests records reference two other BBI records, a Contact and an Organization. While creating your assistance request, you will be asked to select an organization and contact. If they do not exist you will have to leave the Request for Assistance form and create them.
Procedure for creating a request for assistance:

  1. Create the requesting Organization (if it does not exist)
  2. Create the contact person (if it does not exist)
  3. Lastly, complete your assistance request.
